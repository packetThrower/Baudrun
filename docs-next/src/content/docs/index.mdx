## What it does

Baudrun keeps named profiles for each device, runs a full terminal emulator with vendor-aware syntax highlighting, and identifies USB-serial adapters that need a driver. macOS, Windows, and Linux from a single download.

<CardGrid>
  <Card title="Profiles" icon="seti:default">
    Each device gets a saved profile with port, baud rate, parity, line endings, and a theme. Connection takes one click.
  </Card>
  <Card title="Terminal" icon="laptop">
    Built on xterm.js with VT processing, scrollback, search, and syntax highlighting for IPs, MACs, interface names, and vendor-specific output.
  </Card>
  <Card title="Cross-platform" icon="setting">
    macOS, Windows, and Linux from a single Tauri 2 codebase. Native window chrome on each platform, single binary install, automatic updates on stable and pre-release channels.
  </Card>
  <Card title="Adapter detection" icon="seti:powershell">
    Recognizes CH340, CP210x, FTDI, and other common USB-serial chipsets. Identifies missing drivers and links to the vendor download.
  </Card>
</CardGrid>
