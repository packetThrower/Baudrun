## What it does

Baudrun is the console session you actually want when you plug a USB-serial adapter into a switch. Profiles for every device you touch, a real terminal that doesn't fight you, and a UI that gets out of the way once a session is live.

<CardGrid>
  <Card title="Profiles, not flags" icon="seti:default">
    Every device gets a profile with port, baud, parity, line endings, and per-profile theme + highlighting. Click a profile, hit Connect — done. No remembering whether this Cisco wants `8N1` or `7E1`.
  </Card>
  <Card title="Real terminal" icon="laptop">
    Powered by xterm.js with proper VT processing, scrollback, search, and a syntax-highlighting pass for IPs, MACs, interface names, and vendor-specific output.
  </Card>
  <Card title="Cross-platform native" icon="setting">
    macOS, Windows, and Linux from the same Tauri 2 codebase. Native window chrome on each, single binary, no JVM. Auto-update on stable + pre-release channels.
  </Card>
  <Card title="Designed for adapters" icon="seti:powershell">
    Knows about CH340, CP210x, FTDI, and the rest of the USB-serial driver landscape. Tells you when a driver is missing instead of just refusing to open the port.
  </Card>
</CardGrid>
